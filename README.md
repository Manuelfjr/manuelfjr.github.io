# manuelfjr.github.io

Personal website for Manuel Ferreira-Junior.

## Local development

Use Ruby `3.2.2` with `rbenv`:

```bash
export RBENV_VERSION=3.2.2
bundle install
bundle exec jekyll serve --host 127.0.0.1 --port 4000
```

Open `http://127.0.0.1:4000`.

## Python workspace

A local Poetry environment is also available for auxiliary tooling:

```bash
source .venv/bin/activate
```
