## Journal article

### CLAIRE

**Ferreira-Junior, M., Lima Neto, E.A., Ferreira, M.R.P., Silva Filho, T.M., Prudencio, R.B.C.**  
*CLAIRE: clustering evaluation based on item response theory and model agreement.*  
<em>Machine Learning</em>, 114, 256 (2025).

- DOI: [10.1007/s10994-025-06911-0](https://doi.org/10.1007/s10994-025-06911-0)
- Institutional highlight: [CIn UFPE news](https://portal.cin.ufpe.br/2026/02/12/claire-clustering-evaluation-based-on-item-response-theory-and-model-agreement/)

CLAIRE evaluates clustering models through model agreement patterns and item response theory, estimating model ability and instance difficulty in unsupervised settings.

### Evaluating regression algorithms at the instance level using item response theory

**Moraes, J.V.C., Reinaldo, J.T.S., Ferreira-Junior, M., Silva Filho, T., Prudêncio, R.B.C.**  
*Evaluating regression algorithms at the instance level using item response theory.*  
<em>Knowledge-Based Systems</em>, 240, 108076 (2022).

- DOI: [10.1016/j.knosys.2021.108076](https://doi.org/10.1016/j.knosys.2021.108076)

This paper applies item response theory to regression evaluation at the instance level and introduces Gamma-IRT as a way to analyze absolute prediction errors, dataset difficulty, and model ability.

- DOI: [10.48550/arXiv.2303.17731](https://doi.org/10.48550/arXiv.2303.17731)
